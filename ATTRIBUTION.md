## Adaptation note

This repository is not a verbatim copy. It is a localized and restructured adaptation for an OpenClaw-style skill workflow, with:

- a skill-family layout
- reference-file based progressive disclosure
- task shaping, delegation, verification, checkpointing, and delivery-oriented patterns
- Chinese and English packaging for easier reuse

If you reuse this repository, keep the attribution chain intact where appropriate.
